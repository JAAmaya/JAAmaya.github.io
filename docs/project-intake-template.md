## Checklist antes de publicar

- [ ] No incluye teléfono ni datos privados.
- [ ] Los enlaces son públicos o están aprobados para publicación.
- [ ] El cliente/equipo permite mostrar el proyecto.
- [ ] Las imágenes, vídeos o logos pueden usarse públicamente.
- [ ] La contribución personal está clara.
- [ ] La descripción y los highlights no dicen lo mismo.
- [ ] Hay versión corta para CV y versión más completa para portfolio.
